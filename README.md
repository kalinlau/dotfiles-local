# Dotfiles-local
This project contains mac specific dotfile configuration.

## Useful Tools

* `ack`
* `ag`
* `autojump`
* `binutils`
* `coreutils`
* `ffmpeg`
* `font-hack-nerd-font`
* `fzf`
* `htop`
* `imgcat
* `nnn`
* `ncdu`
* `tldr`
* `tmux`
* `youtube-dl`
* `Zsh`

|  |
